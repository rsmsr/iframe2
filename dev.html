A Sample Page for iFrame testing

This is a sample page for testing iFrame, where the PRS form is embedded within the landing page.

To add the PRS form in your iFrame, follow these steps

  • Download "Seamless" javascript package from https://github.com/rsmoorthy/seamless/releases/download/v1.0/seamless_v1.0.min.js and install in your server
  • Include Seamless in your page header, along with jQuery, BootStrap and Bootbox 
    <link rel="stylesheet" href="//cdnjs.cloudflare.com/ajax/libs/twitter-bootstrap/3.3.7/css/bootstrap.min.css" type="text/css">
<link rel="stylesheet" href="//cdnjs.cloudflare.com/ajax/libs/twitter-bootstrap/3.3.7/css/bootstrap-theme.min.css" type="text/css">

<script src="//cdnjs.cloudflare.com/ajax/libs/jquery/3.2.1/jquery.min.js"></script>
<script src="//cdnjs.cloudflare.com/ajax/libs/twitter-bootstrap/3.3.7/js/bootstrap.min.js"></script>
<script type="text/javascript" src="//cdnjs.cloudflare.com/ajax/libs/bootbox.js/4.4.0/bootbox.min.js"></script>


<script src="/js/seamless_v1.0.min.js" type="text/javascript"></script>
(OR)
<script src="https://s3.ap-south-1.amazonaws.com/rsm-seamless/seamless.min.js" type="text/javascript"></script>
  • Include the following script in your html page, at the bottom 
    <script type="text/javascript">
  $(function() {
    var seamless = new Seamless({acceptFrom:['prsdemo.isha.in', 'prs.isha.in']});
  });
</script>
  • If there are more than one iframe element, you need to include the appropriate query selector for the specific iframe, you are interested in. You could do so by: 
    <script type="text/javascript">
  $(function() {
    var seamless = new Seamless({
      acceptFrom:['prsdemo.isha.in', 'prs.isha.in'],
      childElement: '#prs'    // This is the query selector for my iframe
    });
  });
</script>
  • Option 1: Include the iframe pointing to prs form, as below (URL Directly specified) 
    <iframe id="prs" src="https://prsdemo.isha.in/html/forms.html/registration_test1_yaml?_seamless=isha.sadhguru.org"
  scrolling="no" style="overflow: hidden; border: none; height: 80%; width: 100%"/>
  • Option 2: Include the iframe pointing to prs form, but allow parent url params to be copied 
    <iframe 
      id="prs"
      data-src="//prs.isha.in/html/forms.html/registration_test1_yaml"
      data-params="_seamless=isha.sadhguru.org"
      data-copyurlparams="1"
  scrolling="no" style="overflow: hidden; border: none; height: 80%; width: 100%"/>
